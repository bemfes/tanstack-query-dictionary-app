# About Project

This app allows you to search for words and get their meanings using the [dictionary API](https://dictionaryapi.dev/). You can also change the color theme (light and dark).

This project is made using React, React Context, TypeScript, Tanstack Query and Vite.

The project uses commit linting with Сommitlint and Husky. Also, for each commit, linting and code formatting of committed files is performed using lint-staged.

[See the demo](https://bemfes.github.io/tanstack-query-dictionary-app/)
